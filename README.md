# cv-tariska-
Website Curriculum Vitae (CV) berbasis HTML, CSS, Bootstrap, dan JavaScript yang dibuat sebagai proyek Ujian Tengah Semester (UTS) mata kuliah Pemrograman Web. Website menampilkan profil pribadi, riwayat pendidikan, pengalaman kerja, keahlian, bahasa, hobi, galeri kegiatan serta fitur interaktif menggunakan Javascript
